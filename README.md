## Install

Install Node.js version  20.11.1

Install Angular version 18
```sh
npm i @angular/cli
```


install ionic version 8
```sh
npm i @ionic/cli
```


```sh
npm install
```

## Usage

```sh
ionic serve
```

## Run tests

```sh
npm run test
```
